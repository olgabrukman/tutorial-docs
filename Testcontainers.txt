=====================================
Testcontainers
=====================================
Testcontainers is a Java library that supports JUnit tests, providing lightweight, 
throwaway instances of common databases, Selenium web browsers, or anything else 
that can run in a Docker container.

https://www.testcontainers.org/

Repository with lots of examples:
https://github.com/testcontainers/testcontainers-java/tree/master/examples
